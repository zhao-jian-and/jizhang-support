# 隐私政策
竹林记账（以下简称"本应用"）非常重视用户的隐私保护。本隐私政策说明了我们收集、使用、存储和保护用户信息的方式。

## 1. 信息收集

本应用不会收集任何个人身份信息。我们不会要求您注册账户，也不会收集您的姓名、电话、邮箱等个人信息。

应用内功能所需的信息：
- 记账数据：金额、分类、备注、日期等（仅存储在您的设备本地）
- 语音输入：用于AI智能记账功能（语音内容仅在设备端处理，不上传服务器）
- 相机权限：用于OCR扫描记账功能（照片仅在设备端处理，不上传服务器）

## 2. 数据存储

所有记账数据均存储在您的iOS设备本地，不会上传到任何服务器。我们不会备份您的数据，建议您定期使用数据导出功能备份重要数据。

## 3. 数据使用

本应用收集的数据仅用于为您提供记账服务：
- AI智能记账：分析您输入的文字或语音，识别金额、分类等信息
- OCR扫描：识别您拍摄的小票照片，提取商品和价格信息

## 4. 第三方服务

本应用使用以下第三方服务：
- Apple StoreKit：处理应用内购买（订阅专业版）
- Apple Speech Recognition：语音识别功能
- Vision Framework：OCR图像识别功能

以上服务均在您的设备本地执行，不会将您的数据传输给第三方。

## 5. 儿童隐私

本应用不针对13岁以下儿童，也不收集儿童个人信息。

## 6. 政策更新

我们可能会更新本隐私政策。如有重大变更，我们将通过应用内通知的方式告知您。

如果您对本隐私政策有任何疑问，请通过应用内"联系我们"功能与我们沟通或者发送邮件到634884691@qq.com。
